Recursive Random Array Filler (C#)

A C# console program that takes an array size as input and fills the array with random values using recursion.

Features

· Takes array size dynamically from user input
· Fills array with random numbers using recursive method
· Implements Random.Next() for value generation
· Demonstrates recursion with user-defined array length

Technologies

· C# (.NET Console Application)


Usage

1. Run the program
2. Enter the desired array size
3. Program recursively fills and displays the array
